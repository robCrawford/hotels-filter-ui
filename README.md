Hotels Filter UI
================

Hotel results filtering test for TravelRepublic, built with Express.js and Bootstrap.  


To install
----------
`cd /path/to/app/`  
`npm install`  

To run
------
`node app.js`  
Visit `http://localhost:3000` in a browser.  

To test
-------
Back-end: `npm test`  
Front-end: `karma start` (or `npm run test-karma`)
